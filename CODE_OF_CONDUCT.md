# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1, as its code of conduct.

In short: be respectful, assume good faith, and keep the project welcoming for
everyone. The full text, including the standards we hold each other to and how
enforcement works, is at the link above.

## Enforcement

Report unacceptable behavior to the maintainer through
the repository maintainer contact. Reports are reviewed and kept
confidential.
